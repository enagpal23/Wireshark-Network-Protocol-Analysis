# Wireshark-Network-Protocol-Analysis
This project demonstrates my ability to analyse network traffic using Wireshark. I investigated several common network protocols by examining packet captures and identifying how devices communicate across the TCP/IP model.

The project includes protocol analysis, packet inspection, IP addressing, port analysis, and observations of client-server communication for HTTP, SMTP, DNS, and HTTPS traffic.

Note: Some analyses are based on educational packet captures. The original .pcapng files are not included because they were provided as course materials and may not be licensed for public distribution. All analysis, explanations and documentation in this repository are my own work.

- TCP
